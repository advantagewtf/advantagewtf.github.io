SoarClient 7 End Of Life

SoarClient has been discontinued.

Many users have been asking questions like "How do I download?", "Why is the SoarClient site down?", and "Is SoarClient dead?" The answer is simple: you need to use an alternative version like SoarClient Legacy or Advantage.

Thank you for using SoarClient

Your Options

For now, you have two options to continue using SoarClient features:

Actively Maintained

Advantage

Advantage is a fork of Soar developed by several of the SoarClient staff members. It's actively maintained with ongoing support and regular updates. Advantage includes all the features you love about Soar and continues to add new improvements.

Get Advantage
No Longer Updated

SoarClient Legacy

SoarClient Legacy/Offline is SoarClient as it was when v7 was abandoned. It's out of date and no longer maintained. While you can still use it, any bugs found will not be fixed, and no new features will be added. We recommend using Advantage for a better experience.
